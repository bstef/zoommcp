THE CONCEPT

Zoom MCP connects Claude (AI assistant) with Zoom so you can manage 
meetings with simple voice commands. No logging in, no clicking around 
— just talk to Claude about your schedule.

    "Show my upcoming meetings"
    "Create a meeting for tomorrow at 10am"
    "What's the recording from yesterday's standup?"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

HOW IT WORKS

The system creates a secure bridge between Claude and Zoom's API:

    1. Run ./run.sh
    2. Banner/startup mode is displayed for clear status
    3. Your meetings page opens automatically in the browser
    4. Claude Desktop is verified to be running (opened if needed)
    5. Token is refreshed only when expired (or via force mode)
    6. MCP server starts and Claude receives access to manage your Zoom account
    7. Authentication tokens stay fresh automatically (monitored every 60 seconds)
    8. Start asking Claude about your meetings

GETTING STARTED

  1. Grab API credentials from Zoom Marketplace
  2. Add them to your .env file
  3. Run: ./run.sh
  4. Ask Claude about your meetings!

WHAT YOU'LL SEE

When you run ./run.sh, the system will:
  
  ✅ Show startup banner and mode
  🌐 Open Zoom meetings page in your browser
  🔍 Smart startup verification for Claude Desktop
  ✅ Validate your access token (or fetch new one only if expired)
  🚀 Start the MCP server with periodic token monitoring
  🔄 Display token status every 60 seconds (color-coded status)

Output looks like:
  "✅ Claude Desktop is running"
  "✅ Zoom token is still valid. No refresh needed."
  "━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━"
  "✅ Zoom MCP Server is running on stdio"
  "🔄 Token monitoring active - updates every 60 seconds"
  "━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━"
  "🔑 Token Status: ✅ Expires in 57m 29s (at 1:51:23 PM)"

Every 60 seconds you'll see periodic token status updates with separators.
Status indicators:
  ✅ = Fresh token (>15 min remaining)
  📊 = Token expires soon (5-15 min remaining)
  ⚠️  = Token expires very soon (<5 min) - auto-refresh triggers

Now talk to Claude about your Zoom meetings!

UNDER THE HOOD

  • Node.js server bridging Claude ↔ Zoom API
  • Server-to-Server OAuth for security
  • Smart token management (validates before refreshing)
  • Automatic token refresh (never expires mid-use)
  • Claude MCP-safe Node launcher (`launch-mcp.js`) to avoid shell permission issues
  • Cross-platform support (macOS, Linux, Windows)
  • Integrated with Claude Desktop
  • Enhanced CLI output with emojis and user-friendly messages
