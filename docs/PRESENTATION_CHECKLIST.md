## 🎯 During Demo - Troubleshooting

**If command fails:**
- Don't panic, say "Let me try that again"
- If fails twice, move to screenshots
- Say "I tested this 30 minutes ago, but let me show you screenshots"

**If Claude is slow:**
- Talk through what it's doing
- "Claude is querying the Zoom API right now..."
- Use the wait time to emphasize automation

**If network dies:**
- Have backup screenshots
- Say "Let me show you what it looks like when it works"
- Offer to do live demo after meeting
