Key Capabilities

Everything You Need

Smart token management, auto-refresh, and seamless Claude integration with native macOS support.

🔑

Smart Token Management

Validates before fetching. Only requests a new token when expired or within 5 minutes of expiry. Atomic writes for iCloud Drive safety.

🔄

Auto Token Refresh

Background monitor refreshes tokens automatically. Anti-loop protection with 30s cooldown and max 3 failure limit.

🚀

Claude Desktop Integration

Multi-method startup verification with up to 15 retries. Auto-reopens Claude if it closes while the server is running.

🛡️

iCloud-Safe Launcher

launch-mcp.js avoids shell execution restrictions when running from Claude on macOS paths with spaces or iCloud Drive.

📊

Live Status Monitoring

Token expiry countdown updates every 60 seconds with color-coded indicators: ✅ fresh, 📊 expiring soon, ⚠️ critical.

🍎

Native macOS App

Electron app with real-time logs, one-click server controls, token panel, and Claude restart — no terminal required.

Desktop Control

App Features

Everything you need to manage the MCP server and Zoom credentials from your desktop.

🎮

One-Click Server Control

Start and stop the MCP server instantly. Server state is reflected in real time across the status dashboard and badge.

📋

Live Log Viewer

Terminal-style log panel streams server stdout/stderr in real time with color-coded output: green for status, blue for token info, yellow for warnings.

🔐

Token Panel

Displays the current Zoom access token with source attribution (.env or Claude config). Check and refresh from the app without the terminal.

🔁

Restart Claude

Restart Claude Desktop directly from the app with the purple gradient button — useful after token refreshes that require Claude to reload its config.

ChatGPT Apps Integration

Run Zoom MCP as an HTTP server accessible to ChatGPT Apps and other AI clients using OpenAI's protocol.

🌐

HTTP Transport Mode

Set MCP_TRANSPORT=http and MCP_HTTP_PORT to run as a standalone HTTP server instead of stdio.

🔗

ngrok Tunnel

Use ngrok http 3001 to expose your local server to the internet securely. Perfect for ChatGPT Apps and remote access.

📡

Remote Access

Share the public ngrok URL with ChatGPT Apps or other MCP clients. Full OAuth support over HTTPS tunnel.

🔒

Secure by Default

ngrok provides encryption and authentication. All Zoom API calls use OAuth 2.0 Server-to-Server with secure token storage.

Getting Started

Setup & Configuration

Three simple steps to integrate Zoom with Claude.

Environment Variables

Variable Purpose Status
ZOOM_CLIENT_ID Zoom OAuth client identifier Required
ZOOM_CLIENT_SECRET Zoom OAuth client secret Required
ZOOM_ACCOUNT_ID Your Zoom account ID for S2S auth Required
OPENAI_API_KEY Optional OpenAI API key (openai branch) Optional
MCP_TRANSPORT Transport type: stdio or http Optional
MCP_HTTP_PORT HTTP port when using http transport Optional

Claude Desktop Config

~/.claude/claude_desktop_config.json
{ "mcpServers": { "zoom": { "command": "node", "args": ["/path/to/zoommcp/index.js"], "env": { "ZOOM_ACCESS_TOKEN": "<your-access-token>" } } } }
