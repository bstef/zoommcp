Branch Types And Purpose

main

The easiest way to run this project. Focused on stable Claude Desktop usage with token refresh automation and simple startup flow.

easy run path

feature/macos-app

Focused on building and packaging a macOS app (Electron) so the project can be used as a desktop app workflow.

build app path

openai

Focused on ChatGPT integration using Streamable HTTP transport, plus ngrok/public HTTPS flow for ChatGPT Apps connectivity.

chatgpt integration
Exposed MCP Tools
📅

list_meetings / get_meeting

Read Zoom meetings and inspect detailed metadata including join links and schedule.

🛠️

create/update/delete_meeting

Create, modify, or remove meetings directly through MCP tools.

👥

list_users / get_user

Retrieve account users and profile details for scheduling and ownership workflows.

🎬

get_meeting_recordings

Pull recording metadata for completed meetings.

📊

get_meeting_participants

Inspect participant lists and attendance details.

🔑

create_openai_api_key

Helper workflow for OpenAI API key setup in the ChatGPT-focused branch path.
